Social Network Analysis Tool for Moodle

-------------------------------------


===ABOUT===

Social Network Analysis Tool can show the sociogram,density,outdegree,indegree,point centrality of the forum. 

===

===REMARKS===

The implementation of this plugin is in its early stage and it may have problems when it is installed in your Moodle. It was tested in Moodle 2.5
And you must change the max_execution_time in the php.in.The default number is 30,please change it to 10000 or more.
